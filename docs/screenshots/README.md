# Screenshots

Marketing captures for README, Product Hunt, and portfolio.

## How to capture

1. `npm run dev`
2. Open http://localhost:3000
3. Use 1280×800 or 1440×900 viewport (Chrome DevTools)
4. Save PNGs with the filenames below

## Files

| Filename | Section to capture |
| -------- | ------------------ |
| `hero.png` | Hero + sample audit preview card |
| `features.png` | Features grid (`#features`) |
| `workflow.png` | Workflow steps (`#workflow`) |
| `cta.png` | Final CTA section (bottom) |
| `full-page.png` | Full scroll (optional) |

Place files in this directory (`docs/screenshots/`).
